  ```tex
  \begin{itemize}

- [ ] Enumerate
  
  ```tex
  \begin{enumerate}
  \item``` pencil
  \end{enumerate}
  ```

  - [ ] Enumerate using another letter
    ```tex
    \begin{enumerate}[A.]
    ```
  
  > Requires `enumerate` package

     - [ ] Start Numbering from Another Number 
     
     ```tex
    \begin{enumerate} \setcounter{enumi}{6}
     ```
    
    > Start from 7
