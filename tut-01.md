# Soft Enter: ```\\```
  Like a normal Enter
  
  Input:
  ```
  Hello there!\\
  It's a new paragraph.
  ```
  Output: 
  ```
    Hello there!
  It's a new paragraph.
  ```
  
# Hard Enter: ```Insert a Blank Line```
  Like a new paragraph
  
  Input:
  ```
  Hello there!
 
  It's a new paragraph.
  ```
  Output: 
  ```
    Hello there!
    It's a new paragraph.
  ```
